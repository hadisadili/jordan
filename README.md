# Jordan Bennett Portfolio

A minimal portfolio website built with Next.js 14, featuring a clean design with dark/light mode toggle.

## Features

- 🎨 Clean, minimal design
- 🌓 Dark/Light mode toggle (top right)
- 📱 Responsive layout
- 🎬 Chronological list of video work
- 🔗 Direct links to YouTube videos

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React Server Components

## Project Structure

- `/app/page.tsx` - Main portfolio page with video listings
- `/app/components/ThemeToggle.tsx` - Dark/light mode toggle switch
- `/app/globals.css` - Global styles
